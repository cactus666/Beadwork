Используется для точек крепления на базовой верёвке и для крепления в связке среднего участника, за что получил своё второе распространённое название. Для привязывания верёвки к основным и промежуточным креплениям, для временной изоляции дефектного (перебитого) куска рабочей верёвки.

Узел примечателен тем, что легко, по сравнению с восьмёркой, развязывается и одинаково хорошо затягивается в обе стороны. При рывке с одной из сторон не скользит по веревке.
